技术栈

后端 (server/)

  • •Node.js 20 + Express + WebSocket
  • •MySQL 8.0（用户/密文持久化）
  • •Redis（在线状态 + 多节点路由）
  • •Cloudflare R2（S3 兼容）
  • •JWT + bcrypt 认证

前端 (client/)

  • •原生 HTML + Vanilla JS（无构建工具）
  • •libsodium-wrappers（WASM Curve25519 / XSalsa20-Poly1305）
  • •WebRTC（视频/语音通话）
  • •PWA（manifest + Service Worker）

加密层 · 无状态 ECDH + XSalsa20-Poly1305

逐消息临时密钥 · 前向保密
私钥四层持久化（内存 → localStorage → sessionStorage → IndexedDB）
私钥永不离开设备

三种部署方式 · 零配置起步

Docker 一键部署 · Zeabur 云部署 · 本地开发

方式零 · 推荐

Zeabur 一键云部署

无需服务器，点击按钮自动创建完整环境

部署后需手动设置 client 的 SERVER_URL
方式一 · 最简单

Docker Compose

git clone https://github.com/619dev/PaperPhone.git
cd PaperPhone
cp server/.env.example server/.env
docker compose up -d

镜像已发布至 Docker Hub
server 首次启动自动创建数据库

方式二 · 本地开发

本地手动启动

cd server
npm install && npm run dev

# 前端
npx serve client -p 8080

支持热重载 · 适合二次开发

支持多节点 Redis + MySQL · Cloudflare TURN 已内置
